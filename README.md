# SD Assignment 1
## Help guide to run.sh script
### Run once
```bash
./run.sh
```
### Run N times, with N >= 1, (log files stored in a newly created logs directory at root)
```bash
./run.sh N
```

### Clean .class and log files
```bash
./run.sh clean
```

### Check integrity of log file
```bash
./run.sh check-log FILEPATH
```
